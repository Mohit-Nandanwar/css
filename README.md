# Laundry Booking Web Page

This project is a **basic HTML & CSS webpage** designed for a basic user interaction web pages.  


## Features
- Built with pure HTML & CSS  
- Simple booking form structure  
- Easy to customize and extend  

## Usage
1. Clone or download the project files.  
2. Open `index.html` in your browser.   

## Future Improvements
- Connect with backend for real-time booking.
